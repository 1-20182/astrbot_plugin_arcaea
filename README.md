# Arcaea 表情包生成器插件 🎮

> 为 AstrBot 打造的 Arcaea 风格表情包生成器，让你的聊天更加生动有趣！

## 📦 功能特性

- **智能文字排版**：根据文字长度自动调整显示方式，短文字两边渲染，长文字单侧渲染
- **丰富角色选择**：支持 20+ 位 Arcaea 角色，包括光、对立、彩梦等热门角色
- **高度自定义**：可调整字体大小、旋转角度、边距等参数
- **随机表情包**：支持通过关键词触发随机发送表情包
- **双模式生成**：简化模式和复杂模式满足不同需求

## 🚀 快速开始

### 安装方法

1. 将插件文件夹放入 AstrBot 的插件目录
2. 重启 AstrBot 即可自动加载

### 基本用法

- **生成表情包**：`/arc <角色> <文字>`
  - 示例：`/arc 光 你好`（短文字，双文字效果）
  - 示例：`/arc 对立 欢迎光临`（长文字，单文字效果）

- **查看角色列表**：`/arc list`
- **查看图片角色列表**：`/arc角色列表`
- **随机表情包**：`/arc表情包` 或发送包含触发关键词的消息
- **查看帮助**：`/arc_help`

## ⚙️ 配置选项

在 WebUI 插件配置页面，你可以调整以下参数：

| 配置项 | 默认值 | 说明 |
|-------|-------|------|
| `custom_font_path` | `""` | 自定义字体文件路径（复杂模式使用） |
| `simple_font_size` | `35` | 简化模式字体大小（推荐范围 20~45） |
| `simple_left_rotation` | `-45` | 左上角文字旋转角度（逆时针为正） |
| `simple_right_rotation` | `45` | 右上角文字旋转角度（逆时针为正） |
| `simple_left_margin` | `20` | 左上角文字距离左边和上边的像素数 |
| `simple_top_margin` | `20` | 上边距（留空则与左边距相同） |
| `simple_right_margin` | `20` | 右上角文字距离右边和上边的像素数 |
| `trigger_keywords` | `"arc表情包"` | 触发随机表情包的关键词（多个用逗号分隔） |

## 📁 目录结构

```
astrbot_plugin_arcaea_sticker/
├── fonts/           # 字体文件目录
├── list/            # 角色预览图目录
├── resources/       # 角色立绘资源
├── utils/           # 工具类
├── _conf_schema.json # 配置模式定义
├── main.py          # 主插件文件
├── metadata.yaml    # 插件元数据
└── requirements.txt # 依赖项
```

## 💡 小提示

- 如需添加自定义表情包，请将图片放入 `emoji` 文件夹
- 如需添加角色预览图，请将图片放入 `list` 文件夹
- 如需使用自定义字体，请在 `fonts` 文件夹中放入 TTF 格式字体文件
